## Installation

Install directly from GitHub using npm:

```bash
npm install github:0x30/vue-navigation
```

Or using pnpm:

```bash
pnpm install github:0x30/vue-navigation
```

Or using yarn:

```bash
yarn add github:0x30/vue-navigation
```

## Usage

After installation, you can import the packages in your project:

```js
// Import core navigation functionality
import { navigation, push, back } from 'navigation'

// Import Vue components
import { NavPage, SidePage } from 'navigation-vue'
```

For detailed usage examples, see the package READMEs:
- [navigation documentation](packages/core/readme.md)
- [navigation-vue documentation](packages/layout/readme.md)
